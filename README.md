# Үйдегі күтім — MVP «Семейный мост»

Семейный штаб ухода за пожилыми родителями: журнал состояния, задачи между родственниками, эскалация, отчёт для врача, скрипты разговоров.

> ⚠️ Информационный сервис, не замена врача. При угрозе жизни — **103**.

## Возможности MVP

- Создать семейный круг и **код приглашения** (6 символов)
- Присоединиться по коду (координатор / ухаживающий / помощник)
- **Журнал** — быстрые отметки: хороший день, боль, тошнота, слабость и др.
- **Эскалация** — предупреждение, если 3 дня подряд «тревожные» симптомы
- **Задачи** — кто записывает к врачу, покупает лекарства
- **Скрипты** — фразы для разговора с родителями и siblings
- **Отчёт для врача** — сводка за 30 дней, печать из браузера

## Запуск

```bash
cd family-care-mvp
npm install
npm run dev
```

Откройте: **http://localhost:3001**

Порт можно сменить: `PORT=3002 npm run dev`

## Как протестировать за 5 минут

1. Создайте круг (например: «Семья Тест», «Мама», «Дана»).
2. Скопируйте код и откройте в другом браузере / режиме инкогнито.
3. Войдите как брат/сестра по коду.
4. На главной нажмите «Боль» или «Тошнота» три дня подряд — появится баннер эскалации.
5. Вкладка **Врачу** → «Открыть отчёт для печати».

## Стек

- Node.js + Express
- JSON-файл как БД (`server/data/database.json`)
- Vanilla HTML/CSS/JS (PWA-ready, mobile-first)

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/families` | Создать круг |
| POST | `/api/families/join` | Войти по коду |
| GET | `/api/families/:code` | Данные круга |
| POST | `/api/families/:code/symptoms` | Запись в журнал |
| POST | `/api/families/:code/tasks` | Новая задача |
| PATCH | `/api/families/:code/tasks/:id` | Выполнить задачу |
| GET | `/api/families/:code/report` | Отчёт для врача |
| GET | `/api/scripts` | Скрипты разговоров |
